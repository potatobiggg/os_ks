# VFS 测试指南

## 启动

```bash
cd f:/osks/vfs
gcc -std=c11 -g -O0 -o vfs.exe src/*.c
./vfs.exe myfs.dat
```

> 用户密码均为 `123`

---

## 测试 1：登录与密码

```
login usr1 123        # 成功，自动创建 /usr1 并进入
login usr1 wrong      # 失败：密码错误
```

## 测试 2：文件读写

```
login usr1 123
create hello.txt
open hello.txt w
write 0 Hello VFS!
close 0
open hello.txt r
read 0 50              # 读到 "Hello VFS!"
close 0
```

## 测试 3：目录操作

```
login usr1 123
mkdir sub
chdir sub
create data.bin
dir                    # 显示 data.bin
chdir ..
dir /usr1/sub          # 显示 data.bin
```

## 测试 4：用户隔离

```
login usr1 123
create secret.txt
logout
login usr2 123
dir /usr1              # 失败：permission denied
chdir /usr1            # 失败：permission denied
open /usr1/secret.txt r  # 失败：路径被拦截
```

## 测试 5：删除与保护

```
login usr1 123
mkdir testdir
create testdir/a.txt
delete testdir         # 失败：directory not empty
delete testdir/a.txt
delete testdir         # 成功
```

## 测试 6：持久化

```
login usr1 123
create save.txt
open save.txt w
write 0 Survive reboot!
close 0
logout
exit                   # 保存退出
```

重启 `./vfs.exe myfs.dat`：

```
login usr1 123
open save.txt r
read 0 50              # 读到 "Survive reboot!"
```

## 测试 7：格式化

```
format                 # 清空所有数据
login usr1 123         # 重新创建 /usr1
dir /                  # 只有 usr1，之前的文件全消失
```

---

## 命令速查

| 命令 | 格式 | 说明 |
|------|------|------|
| login | `login usr1 123` | 8 用户，密码均 123 |
| logout | `logout` | 注销 |
| create | `create /path/file` | 创建文件 |
| open | `open /path/file r\|w\|rw` | 打开，返回 fd |
| close | `close 0` | 关闭 fd |
| read | `read 0 100` | 从 fd 读 N 字节 |
| write | `write 0 text` | 向 fd 写文本 |
| delete | `delete /path/file` | 删除 |
| mkdir | `mkdir /path` | 创建目录 |
| chdir | `chdir /path` | 切换目录 |
| dir | `dir [path]` | 列出目录 |
| format | `format` | 格式化 |
| exit | `exit` | 保存并退出 |
